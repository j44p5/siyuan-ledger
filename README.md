# SiYuan Ledger · 思源记账本

一个本地优先、数据随笔记同步的轻量记账插件。

## 特性

- 🗒️ **斜杠命令 `/记账`** — 在笔记中直接唤起快速记账，自动关联当前笔记块
- 🧠 **智能语义解析** — 支持自然语言输入（如"买烟40元"、"打车回家花了25"、"工资5000"），自动识别金额、类型、分类、日期和备注
- 📋 **侧边栏 Dock 面板** — 快速记一笔 + 当月收支汇总 + 最近交易记录
- 📊 **完整记账视图** — TopBar 打开全功能对话框，支出/收入/统计/查询四个标签页，支持交易的增删改查
- 📈 **统计图表** — 月度/年度支出环形图 + 近 12 个月收支趋势柱状图（纯 SVG，零依赖）
- 🔍 **多条件查询** — 按日期范围、收支类型、分类、金额区间筛选交易记录，快速定位历史账单
- 🔗 **笔记块关联** — 通过斜杠命令录入的交易自动绑定当前笔记块，支持一键跳转
- 🔒 **数据自动同步** — 通过思源插件 data API 存储，随笔记多端同步
- 🌍 中英双语

## 使用方式

### 斜杠命令（推荐）

在任意笔记中输入 `/记账`，弹出输入框，支持两种格式：

- **自然语言**：直接输入，如 `买烟40元`、`打车回家花了25`、`工资5000`、`充话费100昨天`
- **管道分隔**：`金额|类型|分类|备注|日期`，如 `40|支出|餐饮|买烟`

录入后交易自动保存，并在当前笔记块中插入一条格式化的摘要。

### Dock 面板

右下角 Dock 面板提供快速录入表单、当月收支汇总和最近 20 条交易记录。

### TopBar 完整视图

点击顶部工具栏图标打开完整记账对话框：

- **支出 / 收入**：完整录入表单 + 月度汇总 + 交易列表（支持编辑、删除、跳转到关联笔记）
- **统计**：月度与年度支出环形图、近 12 个月收支趋势柱状图
- **查询**：按日期范围、收支类型、分类、金额区间多条件筛选，支持组合查询

```

├── src/

│       ├── QuickEntry.ts     # Dock 面板快速录入表单

```

## 本地开发

```bash
pnpm install
pnpm dev   # watch 模式编译到 dist/
```

把 `dist/` 拷贝到思源的 `{workspace}/data/plugins/siyuan-ledger/`，在思源「已下载插件」里启用，修改后刷新即可生效。也可以使用 `scripts/dev-deploy.sh` 一键构建部署。

## 数据说明

- 金额一律存「分」(cents) 整数，避免浮点误差
- 日期存 ISO `yyyy-mm-dd`
- 数据文件：`{workspace}/data/storage/petal/siyuan-ledger/ledger.json`
- 默认分类：餐饮、交通、购物、居家、娱乐、育儿、社交、其他（支出）；工资、奖金、其他收入（收入）
- 默认账户：现金、银行卡
